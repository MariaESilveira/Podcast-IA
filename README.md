# Projeto Podcast Gerado por I.A.s
O projeto tem o objetivo de gerar um podcast utilizando ferramentas de IA através de prompts mais trabalhados.  
Utilizar uma esteira de prompts para gerar cada etapa do processo criativo e de desenvolvimento do podcast.

## Tecnologias utilizadas para o projeto
* [ChatGPT](https://chatgpt.com/) - Nome e Roteiro do episódio  
* [Stable Diffusion](https://stabledifffusion.com/tools/ai-image-generator#google_vignette) - Capa  
* [ElevenLabs](https://elevenlabs.io/) - Criação do áudio   
* [CapCut](https://www.capcut.com/my-edit?from_page=landing_page&start_tab=video) - Edição do áudio
